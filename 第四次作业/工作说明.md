```python
def call(self, enc_ids, dec_ids):
    # Encoder处理
    enc_emb = self.embed_layer(enc_ids)
    enc_out, enc_state = self.encoder(enc_emb)

    # Decoder处理
    dec_emb = self.embed_layer(dec_ids)
    dec_out, dec_state = self.decoder(dec_emb, initial_state=enc_state)

    # Attention计算
    scores = tf.matmul(dec_out, enc_out, transpose_b=True)
    attn_weights = tf.nn.softmax(scores, axis=-1)
    context = tf.matmul(attn_weights, enc_out)

    # 结合context和decoder输出
    combined = tf.concat([dec_out, context], axis=-1)
    logits = self.dense(combined)
    return logits
```

## 📚 理论说明

### **Seq2Seq模型**

Seq2Seq（Sequence to Sequence）模型采用Encoder-Decoder架构：
- **Encoder**：将输入序列编码为一个固定长度的向量表示
- **Decoder**：根据这个向量表示逐步生成输出序列
- **应用**：机器翻译、文本摘要、对话系统等

### **注意力机制**

注意力机制允许Decoder在生成每个输出时动态关注Encoder的不同部分：

**核心思想**：
- 计算查询（Query，Decoder当前状态）与键（Key，Encoder所有输出）的相关性
- 得到注意力权重（概率分布）
- 对值（Value，Encoder输出）进行加权求和，得到上下文向量
- 将上下文向量与Decoder状态结合，预测下一个输出

**优势**：
- 解决了信息瓶颈问题
- 能够处理长序列
- 提供了可解释性（可视化注意力权重）
